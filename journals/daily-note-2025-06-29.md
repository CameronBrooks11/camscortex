## Design and Knowledge Flow Ideation

### Processing

- Can choose and customize formatting
- allows for assisted / guided processing where the data labels, boxes etc are shown and can be edited or fixed
- next it allows for generative text to edit or add

### Network Architecture

- companies / organizations can use their own server or ours (both cost money thru some mechanism)
- consumers have monthly fee for servers or free for self-hosted
- include api and cmd line
- people can add their paid LLMs (commerical calls) or use free ones (smaller models)

### Digital Inputs

- Any file type (or internet link?) of any format
  - typical processing if possible
  - conversion first if needed (OCR, audio to text, video to text etc)
- Toolchain using default open source apps
  - modular, can swap tools in and out
  - can add custom tools

### Writing Input Surface Idea

- sheet press of two different polymers
- pen with erasable ink
- use some kind of mold for paper finish on plastic "paper"
- forms: notepad, notebook, tabletops, wall sheets
- capture with personal phone camera app or maybe dedicated scanner hardware for real time capture and sharing
