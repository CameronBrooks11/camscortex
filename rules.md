# Rules

- Use the tasks.json file to run tasks.
  - Use `Ctrl+Shift+P` and type "Run Task" to see available tasks.
    - `Update journal index`
- All file names for page entries should have upper case titles with spaces, e.g. "My File Name.md"
- All files names for high-level organizational files should be lower case with hyphens, e.g. "my-file-name.md"
